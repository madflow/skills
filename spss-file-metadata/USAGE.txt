WHAT THE SCRIPT DOES:
---------------------
- Reads SPSS .sav files using pyreadstat with metadataonly=True
- Extracts comprehensive metadata (no data rows loaded)
- Outputs JSON to stdout or saves to file
- Already configured and tested!

DO NOT:
-------

❌ Install pyreadstat again (already installed in venv)
❌ Write inline Python code to read the .sav file

✅ Pass the .sav file path as argument
